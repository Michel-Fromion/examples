Traffic Light ported to Arduino

Based on the standard traffic-light example, this project shows how to use a statechart to develop software for the Arduino platform.

More information about the statechart itself can be found in the standard traffic light example, which should be considered first. In summary, we have two traffic lights here: one for the traffic, and one for pedestrians who want to cross the street. They can push a button to request their traffic light to turn green, which activates a blinking wait-light first. The cars' traffic light then goes through the standard procedure, turning on the yellow, then the red light, allowing the pedestrians to cross safely. After some time, the pedestrians traffic light turns red again, and the cars are allowed to pass until the request is made another time by a pedestrian. Also, the traffic light is able to be switched off completely, which turns the yellow lights into a blinking mode, signaling that the traffic light is indeed switched off and can be crossed in a careful manner.

Install the Arduino plugin

To compile the generated C code for Arduino, you should install an Eclipse plugin that can handle this and brings the needed libraries. On Windows, if you have not already, you should also install the regular arduino IDE, because Windows is lacking the drivers to use the Arduino without it.

One plugin that works pretty well can be found at http://eclipse.baeyens.it/. In YAKINDU SCT, simply install it by going to Help -> Install New Software, paste http://www.baeyens.it/eclipse/V3 into the given field, and follow the regular procedure. SCT will restart and download the needed Arduino libraries for you.

Build the hardware

Get your parts ready! You will need:

  • 2 Pushbuttons
  • 6 LEDs, ideally:
    • 2x red
    • 2x green
    • 1x yellow
    • 1x white
  • 8 Resistors:
    • 2x 10kR or similar
    • 6x 220R or similar

Of course, you can change the pins, but since this represents the pin numbers in the software don't forget to change those as well.

Build the project

With this example in your workspace and active, the Arduino plugin installed, the Arduino connected and the correct port set in Project -> Properties -> Arduino, you can now build and upload the example to your Arduino.

You should have seen the new buttons in your SCT installation: Use the button second to the left to build and upload the code in one step.
