Conventions For Examples

  • An example's title should describe the example as concise as possible. It must occur identically at the following places:

    • as the <title> element in the index.html file

    • as the first (and only) <h1> element in the index.html file

    • as the title attribute in the metadata.json file

  • Possible categories

    • open source
    • basic
    • arduino
    • java
    • c
    • c++
